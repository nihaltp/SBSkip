SB Skip helps you remove SponsorBlock-marked segments from videos and audio files you already have on your device.

Many apps can download media, but the resulting files still contain sponsored messages, self-promotion, interaction reminders, intros, outros, and other segments that SponsorBlock users have identified. SB Skip cleans those files after download, creating a new version with unwanted sections removed.

Features:

• Remove SponsorBlock segments from existing video and audio files
• Supports videos and audio shared from other apps
• Paste a YouTube URL to retrieve SponsorBlock segment information
• Queue multiple cleaning jobs for background processing
• Configure which SponsorBlock categories should be removed
• Persistent queue with progress tracking and notifications
• Separate settings for output behavior and file handling
• Material 3 user interface
• Translation-ready and fully open source

Supported SponsorBlock categories include:

• Sponsor segments
• Self-promotion
• Interaction reminders (like, subscribe, etc.)
• Intros
• Outros
• Preview/recap sections
• Filler content
• Music off-topic sections
• And other categories provided by SponsorBlock

How it works:

1. Select or share a media file with SB Skip.
2. Provide the corresponding YouTube URL.
3. SB Skip retrieves SponsorBlock segment data.
4. Unwanted segments are removed from the media file.
5. A cleaned copy is saved to your device.

SB Skip does not download videos. It is designed to work with media files you already possess and uses the community-maintained SponsorBlock database to determine which sections should be removed.

SB Skip is free software and respects user choice by allowing complete control over which segment categories are processed.
